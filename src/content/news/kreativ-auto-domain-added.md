<a class="cta" href="https://kreativauto.com" target="_blank" rel="noopener">kreativauto.com</a> is now part of the KREATIV ecosystem, giving the automotive project a dedicated place for practical content, references, and future workflow-oriented tools.

The dedicated domain separates automotive content from the creative-tool properties while still keeping it visible in the wider Made by Kreativ directory. That makes the roadmap easier to understand and avoids forcing unrelated topics into the core tool lineup.

Kreativ Auto remains a roadmap-stage project, so the directory listing focuses on direction, status, and the type of practical references the property is expected to support.
